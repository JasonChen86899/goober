# Goober

Goober is the data structure library of [Go](https://golang.org/).
It provides a set of data structure to help developer code better.

## Data Structure

| Data Structure | Concurrency Safe |
|----------------|------------------|
| Cache          | Safe             |
| LRUCache       | Safe             |
| SynMap         | Safe             |
| Ring           | UnSafe           |
| LockFreeRing   | Safe             |
| Channel        | Safe             |
| SkipList       | UnSafe           |
